# Retail Sales Performance & Customer Insights

## Project Overview
This project analyzes retail sales data using **SQL and Power BI** to uncover business insights such as regional profitability, product performance, and sales trends.

The goal is to transform raw transactional data into a **structured data model and interactive dashboard** that helps decision-makers understand performance and identify opportunities.

---

## Tools & Technologies
- **PostgreSQL** – Data storage and SQL analysis  
- **SQL** – Data cleaning, transformation, and aggregation  
- **Power BI** – Data visualization and dashboard creation  

---

## Data Model
A **star schema** was designed to organize the dataset efficiently.

- **Fact Table:** `orders`
- **Dimension Tables:** `customers`, `products`, `geography`

<img src="images/data_model.png" width="800">

This structure improves query performance and simplifies business analysis.

---

## Power BI Dashboard
The interactive Power BI dashboard provides an overview of sales performance.

<img src="images/dashboard.png" width="900">

### Dashboard Features
- **Key Performance Indicators**
  - Total Sales
  - Total Profit
  - Profit Margin

- **Sales Trend Analysis**
  - Monthly sales performance

- **Regional Performance**
  - Profit distribution by region

- **Product Profitability**
  - Identification of loss-making product categories

- **Filters**
  - Year
  - Region

---

## SQL Analysis
SQL queries were used to explore and analyze the dataset.

<img src="images/sql_analysis.png" width="800">

Examples of analysis performed:
- Total revenue and profit calculation
- Monthly sales trend analysis
- Regional profit comparison
- Loss-making product identification
- Customer value analysis

---

## Key Insights
- **West region generates the highest profit** among all regions.
- **Tables and Bookcases are loss-making product categories.**
- The overall **profit margin is approximately 12%.**
- Sales show **strong growth toward the fourth quarter**, indicating seasonal demand.
